# 🍽️ QrOrder

## Çok İşletmeli QR Restoran Sipariş Sistemi

ASP.NET Core ile geliştirilmiş, restoranların QR kod üzerinden sipariş almasını sağlayan çok işletmeli (Multi-Tenant) restoran otomasyon sistemidir.

> ⚠️ Bu proje aktif olarak geliştirilmektedir. Ticari amaçla geliştirildiği için kaynak kodu herkese açık olarak paylaşılmamaktadır.

##  Özellikler

- Çok İşletmeli (Multi-Tenant) yapı
- QR Menü Sistemi
- Gerçek zamanlı sipariş yönetimi (SignalR)
- Mutfak Paneli
- Servis Paneli
- Yönetici Paneli
- Super Admin Paneli
- JWT ile Kimlik Doğrulama
- Rol Bazlı Yetkilendirme
- Ürün ve Kategori Yönetimi
- Sipariş Takibi
- Mobil uyumlu arayüz


## 🛠️ Kullanılan Teknolojiler

- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- SignalR
- JWT Authentication
- HTML
- CSS
- Bootstrap
- JavaScript
